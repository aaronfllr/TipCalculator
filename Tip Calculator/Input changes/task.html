Description

Now that your application has the appropriate UI components, it needs to be able to tell when the user input changes: this way, it will know when to do the calculation and show the tip amount. For now, we will use the EditText field to show the user input values.

Objectives
  • Add an event listener to edit_text.
    • If the value is greater than zero display the value on bill_value_tv with the following format: Bill Value: $[amount], where [amount] is a placeholder for a number always displayed with two decimal places;
    • Also if the value is greater than zero display the current tip percent on tip_percent_tv with the following format: Tip: [tipPercent]%, where [tipPercent] is a placeholder for an integer number. The tip percent value should be equal to the progress attribute on seek_bar
  • Add an event listener to seek_bar
    • When the value of progress attribute changes update the value displayed at tip_percent_tv using the same format described above;
